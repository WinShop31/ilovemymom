# FPS Shooter - 3D онлайн шутер без сервера

Онлайн 3D шутер, работающий прямо в браузере. Мультиплеер через BroadcastChannel (без сервера).

## 🎮 Управление
- **WASD** — движение
- **Пробел** — прыжок
- **Мышь** — обзор
- **ЛКМ** — стрельба

## 🚀 Как играть

### Мультиплеер в одном браузере
1. Открой `index.html` в браузере
2. Открой **ещё одну вкладку** с этой же страницей
3. Игроки увидят друг друга автоматически!

### ⚠️ Ограничения
- Мультиплеер работает **только в одном браузере** (разные вкладки)
- Для игры с разных устройств/компьютеров нужен отдельный сервер

## 📦 Технологии
- **Three.js** — 3D рендеринг
- **BroadcastChannel API** — мультиплеер без сервера
- **PointerLockControls** — управление камерой

## 🔧 Локальный запуск для разработки
```bash
python -m http.server 8080
```
Открой `http://localhost:8080`
